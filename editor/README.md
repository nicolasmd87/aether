# Editor Support

This directory contains editor configurations and language support files for various editors.

## VS Code / Cursor

See `vscode/README.md` for installation instructions.

Language support files are kept here (not in `.vscode/`) to:
- Avoid interfering with users' workspace settings
- Allow the files to be committed to the repository
- Make it clear these are optional editor enhancements

## Future

Additional editor support can be added here:
- `vim/` - Vim syntax files
- `emacs/` - Emacs mode files
- `sublime/` - Sublime Text syntax files
- etc.
